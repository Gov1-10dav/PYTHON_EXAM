### 5. Визуализация в Python (matplotlib, plotly)
---
#### Matplotlib
**Matplotlib — основная библиотека Python для построения графиков.**

Позволяет создавать:
- линейные графики;
- столбчатые диаграммы;
- гистограммы;
- круговые диаграммы;
- точечные графики;
- несколько графиков на одном рисунке.

Основной модуль:
```
import matplotlib.pyplot as plt
```
Пример:
```
plt.plot(x, y)
plt.xlabel("X")
plt.ylabel("Y")
plt.title("График")
plt.show()
```
Matplotlib широко используется совместно с NumPy и Pandas.

---
#### Plotly
**Plotly — библиотека для создания *интерактивных* графиков.**

Её преимущества:
- масштабирование;
- наведение курсора;
- интерактивные легенды;
- красивые графики без большого количества настроек.

Позволяет строить:
- линейные графики;
- столбчатые диаграммы;
- круговые диаграммы;
- тепловые карты;
- 3D-графики;
- карты;
- дашборды.

Plotly часто используется для веб-приложений и анализа данных. Особенно удобен при работе с большими наборами данных, где нужно быстро «погружаться» в конкретный сегмент.
Аналог `seaborn` — высокоуровневый интерфейс над Matplotlib, ориентированный на статистические графики.

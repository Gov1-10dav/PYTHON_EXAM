# 7. Корреляционный и регрессионный анализ в Python (scipy, statsmodels)

## Корреляционный анализ

### Основные понятия

**Корреляция** — количественная мера взаимосвязи между переменными.
- **Прямая** (+): обе переменные растут/убывают вместе.
- **Обратная** (−): одна растёт — другая убывает.
- **Нулевая**: нет зависимости.

> Корреляция ≠ причинно-следственная связь!

---

### Коэффициент корреляции Пирсона

Мера **линейной** зависимости между двумя количественными переменными:

```
r = Σ((xi - x̄)(yi - ȳ)) / sqrt(Σ(xi - x̄)² · Σ(yi - ȳ)²)
```

Интерпретация |r|:
- 0.0–0.2 — слабая
- 0.2–0.5 — умеренная
- 0.5–0.7 — заметная
- 0.7–0.9 — сильная
- 0.9–1.0 — очень сильная

**Условия применения:** оба признака количественные, нормальное распределение, нет выбросов, зависимость линейная.

```python
import numpy as np
import scipy.stats as stats

x = np.array([10, 14, 21, 23, 27, 32, 39, 45, 55, 61, 62, 68])
y = np.array([3.8, 4.8, 5.9, 6.1, 6.2, 6.3, 6.6, 7.4, 8.5, 9.7, 10.5, 12.4])

alpha = 0.05

# Коэффициент корреляции и p-value
r, p_value = stats.pearsonr(x, y)
print(f"r = {r:.4f}, p-value = {p_value:.6f}")

# Проверка значимости через t-критерий
n = len(x)
t_score = r * (n - 2)**0.5 / (1 - r**2)**0.5
t_critical = stats.t.ppf(1 - alpha/2, df=n - 2)

print(f"t = {t_score:.3f}, t_crit = {t_critical:.3f}")
if abs(t_score) > t_critical:
    print("Связь статистически значима")
else:
    print("Нельзя отклонить H0")

# Через p-value:
if p_value < alpha:
    print("Связь значима (p < 0.05)")
```

---

### Ранговая корреляция Спирмена

Используется когда данные **порядковые** или есть выбросы/ненормальное распределение.

```python
rho, p_value = stats.spearmanr(x, y)
print(f"Spearman rho = {rho:.4f}")
```

## Регрессионный анализ

### Основные понятия

**Регрессия** — предсказание количественной зависимой переменной (Y) по независимым переменным (X).

**Допущения линейной регрессии:**
- Переменные близки к нормальному распределению (тест Шапиро-Уилка).
- Линейная связь между X и Y.
- Отсутствие мультиколлинеарности (независимые переменные не зависят друг от друга).
- Отсутствие автокорреляции остатков (тест Дурбина-Уотсона).
- Гомоскедастичность (равенство дисперсий остатков).

---

### Парная линейная регрессия

Уравнение: **y = b₀ + b₁x**

Коэффициенты ищутся методом наименьших квадратов (МНК) — минимизация суммы квадратов отклонений.

```python
import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt

x = np.array([2.2, 2.8, 3.0, 3.5, 3.9, 4.2, 4.4, 4.8, 5.1, 5.5])
y = np.array([0.13, 0.21, 0.28, 0.26, 0.23, 0.29, 0.28, 0.24, 0.31, 0.33])

alpha = 0.05

# Корреляция
r, p_value = stats.pearsonr(x, y)
print(f"r = {r:.3f}, p = {p_value:.4f}")

# Линейная регрессия
res = stats.linregress(x, y)
print(f"slope = {res.slope:.3f}, intercept = {res.intercept:.3f}")
print(f"R² = {res.rvalue**2:.3f}, p = {res.pvalue:.4f}")
print(f"Y = {res.intercept:.3f} + {res.slope:.3f} * X")

# Прогноз
y_pred = res.intercept + res.slope * x

# Доверительный интервал для прогноза в точке x0
x0 = 7
y0 = res.intercept + res.slope * x0
t_coef = abs(stats.t.ppf(alpha/2, len(x) - 2))
s_r = ((y_pred - y)**2).sum() / (len(x) - 2)
delta = s_r**0.5 * (1 + 1/len(x) + (x0 - x.mean())**2 / (len(x) * x.var()))
print(f"При x0={x0}: y0 = {y0:.3f} ± {t_coef * delta:.3f}")

# Визуализация
plt.scatter(x, y, label='Данные')
plt.plot(x, y_pred, 'r--', label='Регрессия')
plt.legend()
plt.show()
```

---

### Тест Дурбина-Уотсона (автокорреляция остатков)

```python
from statsmodels.stats.stattools import durbin_watson

residuals = y - y_pred
dw = durbin_watson(residuals)
print(f"Дурбин-Уотсон: {dw:.4f}")
# ~2 — нет автокорреляции; <1 — положительная; >3 — отрицательная
```

Гипотезы:
- **H₀:** остатки не автокоррелированы (ρ = 0)
- **H₁:** остатки автокоррелированы

---

### Тест Шапиро-Уилка (нормальность)

```python
stat, p = stats.shapiro(residuals)
print(f"Шапиро-Уилк: W={stat:.4f}, p={p:.4f}")
if p > 0.05:
    print("Нормальное распределение (H0 не отвергается)")
```

---

### Тест Гольдфельда-Квандта (гомоскедастичность)

```python
from statsmodels.stats.diagnostic import het_goldfeldquandt

f_stat, p_value, _ = het_goldfeldquandt(residuals, x.reshape(-1, 1))
print(f"Голдфельд-Квандт: F={f_stat:.3f}, p={p_value:.4f}")
# H0: дисперсия остатков постоянна (гомоскедастичность)
```

---

### OLS через statsmodels (полный вывод)

```python
import statsmodels.api as sm

X = sm.add_constant(x)  # добавить столбец единиц для intercept
model = sm.OLS(y, X)
result = model.fit()
print(result.summary())  # полная статистика: коэф., R², F-тест, ...
```

| Показатель | Что показывает |
|-----------|---------------|

### **2. Файлы и файловая система в Python (os)**
---
#### КРАТКО
**Встроенная функция `open()` работает с _содержимым_ файлов, а модуль `os` управляет _файловой системой_ (папки, пути, настройки ОС).** 
#### 1. Работа с файлами (`open`)
- **Открытие:** `open('путь', 'режим', encoding='utf-8')`:
    `r` (чтение),
    `w` (запись/полная перезапись), 
    `a` (дозапись в конец), 
    `r+` (чтение + перезапись).
- **Чтение:** 
	`read()` (всё содержимое)
	`readline()` (одна строка)
	`readlines()` (список всех строк)
- **Запись:** 
	`write()` (текст) 
	`writelines()` (список строк)
	`print(..., file=f)` (удобная запись с автопереносом).
- **Менеджер контекста (`with`):** `with open(...) as f:` → **золотой стандарт**. Гарантирует автозакрытие файла после выхода из блока, даже при ошибках. Ручной `.close()` забывают.
---
#### 2. Модуль `os` (Управление системой)
- **Навигация:** 
	`getcwd()` (текущая папка)
	`chdir()` (перейти в другую)
	`listdir()` (список файлов в папке).
- **CRUD операции:** 
	`mkdir()` (одна папка)
	`makedirs()` (дерево вложенных папок)
	`rmdir()` (удалить _пустую_ папку)
	`remove()` (удалить файл), `rename()` (переименовать/переместить).
- **Метаданные:** 
	`path.getsize()` (размер в байтах)
	`path.getmtime()` (дата изменения)
	`path.abspath()` (полный путь от корня диска).
---
#### 3. Пути и массовый поиск (`os.path` + `walk`)
- **Безопасные пути:** 
	`os.path.join()` (склейка путей с учетом ОС)
	`basename()` (вытащить имя файла)
	`dirname()` (вытащить путь к папке).
- **Проверки:** 
	`exists()` (есть ли объект?)
	`isfile()` (это файл?)
	`isdir()` (это папка?).
- **Рекурсивный обход:** `os.walk()` → проходит по **всем** вложенным папкам. Возвращает кортеж: `(текущий_путь, список_папок, список_файлов)`. Главный инструмент для поиска файлов по расширению или массовых операций.
---
#### 4. Системные вызовы
- `os.system()` → запуск простых консольных команд. Для сложных задач (захват вывода, передача аргументов) в современном Python используют модуль `subprocess`.
---
**Как запомнить для билета:**
1. `open` + `with` = работа с текстом внутри файла (автозакрытие!).
2. `os` = навигация, создание/удаление папок и файлов.
3. `os.path` = безопасная склейка путей и проверки (`exists/isfile`).
4. `os.walk` = рекурсивный "пылесос" по всем вложенным папкам.
---
#### ПОДРОБНО

В Python для работы с файлами и файловой системой используется встроенный инструментарий: базовая функция `open()` для обработки содержимого файлов и мощный встроенный модуль `os` для навигации и управления элементами файловой системы.

---
#### 1. Работа с содержимым файлов в Python

Процесс взаимодействия с файлом обычно состоит из трех шагов: **открытие**, **обработка (чтение/запись)** и **закрытие**.
##### Открытие
Для открытия используется функция `open()`, которая создает файловый объект (дескриптор):

Python
```
файловая_переменная = open(имя_файла, режим_доступа, encoding='utf-8')
```
Указание кодировки `encoding='utf-8'` крайне важно при работе с текстом на русском языке.

***Основные режимы доступа:**
- `'r'` — только чтение (по умолчанию).
- `'w'` — запись (существующий файл стирается, новый — создается).
- `'a'` — добавление (запись данных в конец файла).
- `'r+'` — чтение и запись одновременно (частичная перезапись).
---
##### Чтение данных
Для чтения текстовых файлов используются три метода:

-  `read()` — считывает весь файл в одну строку.
- `readline()` — считывает одну строку (включая символ переноса `\n`).
- `readlines()` — считывает все строки и возвращает их в виде списка.

При чтении перемещается внутренний курсор программы. Его текущее положение можно узнать методом `tell()`, а переместить обратно в начало файла — методом `seek(0)`.

##### Запись данных
- `write(строка)` — записывает строку в файл (не добавляет `\n` автоматически).
- `writelines(список_строк)` — записывает список строк.
- Функция `print(..., file=открытый_файл)` — также позволяет писать в файл, автоматически добавляя перенос строки.
---
##### Менеджер контекста (`with`)
Чтобы не закрывать файлы вручную методом `close()` и гарантировать их закрытие даже в случае ошибок, используют конструкцию `with` (менеджер контекста):

Python
```
with open('languages.txt', 'r', encoding='utf-8') as file:
    for line in file:
        print(line.rstrip())  # rstrip() удаляет лишние переносы строк \n
# Здесь файл уже автоматически закрыт!
```

#### 2. Модуль `os` — управление файловой системой

Модуль `os` предоставляет интерфейс для взаимодействия с операционной системой.
##### Навигация по директориям
- `os.getcwd()` — узнать текущую рабочую папку (аналог `pwd`).
- `os.listdir(путь)` — возвращает список всех файлов и папок внутри указанной директории.
- `os.chdir(путь)` — сменить текущую рабочую папку (аналог `cd`).
---
##### Создание, удаление и переименование
- `os.mkdir('папка')` — создает одну папку.
- `os.makedirs('папка1/папка2')` — создает дерево вложенных папок.
- `os.rmdir('папка')` — удаляет папку, но **только если она пустая**.
- `os.remove('файл.txt')` — удаляет файл навсегда (мимо Корзины!).
- `os.rename('старое_имя', 'новое_имя')` — переименовывает или перемещает файл/папку.
---
##### Получение информации (метаданных)
- `os.path.getsize('файл.txt')` — возвращает размер файла в байтах.
- `os.path.getmtime('файл.txt')` — время последней модификации.
- `os.path.abspath('файл.txt')` — возвращает полный (абсолютный) путь к файлу.
---
#### 3. Подмодуль `os.path` — безопасная работа с путями

В Windows пути разделяются обратным слэшем (`\`), а в Linux и macOS — прямым (`/`). Чтобы код работал на любой ОС, нужно использовать `os.path`:
- `os.path.join('папка1', 'папка2', 'файл.txt')` — корректно собирает путь с учетом ОС.
- `os.path.basename(полный_путь)` — извлекает только имя файла (например, `report.docx`).
- `os.path.dirname(полный_путь)` — извлекает только путь к директории.

**Функции проверки (возвращают `True`/`False`):**
- `os.path.exists(путь)` — существует ли такой файл или папка.
- `os.path.isfile(путь)` — является ли объект существующим файлом.
- `os.path.isdir(путь)` — является ли объект существующей папкой.
Python
```
import os

path_to_check = 'archive'
if not os.path.exists(path_to_check):
    os.makedirs(path_to_check)  # Создаем, если не существует
```
---
##### Рекурсивный обход папок (`os.walk`)
Если нужно пройтись по всем вложенным папкам и найти определенные файлы, незаменима функция `os.walk()`:
Python
```
for root, dirs, files in os.walk("my_project"):
    print(f"Сейчас в папке: {root}")
    for file in files:
        if file.endswith('.txt'):
            print(f"Найден текстовый файл: {os.path.join(root, file)}")
```

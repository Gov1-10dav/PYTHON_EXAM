### 6. Проверка статистических гипотез в Python (scipy)
---
#### **SciPy — библиотека для научных вычислений, построенная на основе NumPy.**

Она содержит инструменты для:
- статистики;
- оптимизации;
- линейной алгебры;
- обработки сигналов;
- интегрирования;
- решения дифференциальных уравнений.

Для проверки гипотез используется модуль:
```
from scipy import stats
```
---
#### Статистическая гипотеза
Статистическая гипотеза — предположение о свойствах генеральной совокупности.

Различают:
- **нулевую гипотезу (H₀)** — различий нет (корреляции нет);
- **альтернативную гипотезу (H₁)** — различия существуют (корреляция есть).

**Этапы проверки гипотезы включают:**
1. Формулировку гипотез H₀ и H₁.
2. Выбор статистического критерия.
3. Расчет статистики и **p-value**.
4. Сравнение p-value с уровнем значимости α (обычно 0,05).
5. Вывод:
    - если **p < 0,05**, H₀ отвергается;
    - если **p ≥ 0,05**, оснований отвергнуть H₀ нет (но это не значит, что гипотеза верна).

**Ошибки:**
- **I рода**: отвергли верную H₀ (вероятность = α).
- **II рода**: не отвергли неверную H₀ (вероятность = β).
- **Мощность критерия** = 1 − β (чем больше — тем лучше)

---
#### Основные статистические тесты SciPy
- **t-критерий Стьюдента (независимый t-тест)**
	Для сравнения средних двух выборок.
	`stats.ttest_ind(x, y)`

- **F-тест (сравнение дисперсий)**
	Проверяет, равны ли дисперсии двух выборок. Используется перед независимым t-тестом (t-критерий Стьюдента):

- **Парный t-тест**
	Для связанных выборок (одна группа, два замера).
	`stats.ttest_rel(x, y)`

- **χ²-критерий (хи-квадрат)**
	Применяется в двух случаях:
		**Тест на соответствие** — проверяет, соответствует ли распределение теоретическому.
		**Тест на независимость** — проверяет связь между двумя категориальными переменными.
	`stats.chi2_contingency(table)`

- **Корреляция**
	Коэффициент Пирсона
	`stats.pearsonr(x, y)`
	Показывает силу линейной связи между признаками.

#### Интерпретация результата
Большинство функций SciPy возвращают:
- значение статистики;
- **p-value**.
Именно по p-value принимается решение о принятии или отклонении нулевой гипотезы.

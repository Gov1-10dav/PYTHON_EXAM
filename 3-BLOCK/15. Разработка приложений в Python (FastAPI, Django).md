## Разработка приложений в Python (FastAPI, Django)

веб-разработка - создание программ, которые работают на сервере и отвечают на запросы сервера. две популярные библиотеки для веб-разработки на python (fastapi & Django) предлагают разные подходы к решению этой задачи

#### основные отличия
| критерий | django | fastapi |
| --- | --- | --- |
| философия | все идет в комплекте | минимализм - инструменты выбирает разраб |
| назначение | полноценные вебсайты, админ-панели, сложные проекты | высокопроизводительные rest api, микросервисы, мобильный бекэнд | 
| работа с бд | встроенная мощная бд | любая внешняя |

django - огромный фреймворк, в который включено множество вещей по умолчанию. он мощный, на нем удобно разрабатывать полноценные проекты (например, множество популярных веб-приложений разработано на django, из примеров помню только спотифай но бля сами поищите я ебу). fastapi - довольно молодой фреймворк, который жертвует масштабом и удобством разработки в пользу невероятной скорости работы. его задача - принимать и отдавать чистые данные

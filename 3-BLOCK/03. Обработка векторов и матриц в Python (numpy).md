### 3. Обработка векторов и матриц в Python (numpy)
---
##### **NumPy (Numerical Python)** — библиотека для научных вычислений, предназначенная для работы с многомерными массивами (векторами, матрицами и тензорами). Она широко применяется в математике, статистике, машинном обучении и анализе данных.

Основной объект библиотеки — **ndarray** (многомерный массив).
```
import numpy as np
```
##### **Преимущества NumPy**
По сравнению со списками Python массивы NumPy имеют:
- более компактное хранение данных;
- высокую скорость вычислений;
- поддержку математических операций над целыми массивами;
- удобную индексацию и срезы;
- большое количество функций линейной алгебры, статистики и обработки данных.

---
#### Создание массивов
```
a = np.array([1, 2, 3])                     # из списка
b = np.zeros((3, 4))                          # нули
c = np.ones((2, 3))                           # единицы
d = np.arange(0, 10, 2)                      # [0, 2, 4, 6, 8]
e = np.linspace(0, 1, 5)                     # 5 равномерных точек от 0 до 1
f = np.eye(3)                                  # единичная матрица
g = np.random.randint(1, 10, (3, 3))         # случайные целые числа
```
Функции позволяют быстро создавать массивы нужной формы и заполнения.
Основной способ создания массива **из списка**.

---
#### Свойства массивов
Основные атрибуты:
- `shape` — размеры массива;
- `size` — количество элементов;
- `dtype` — тип данных;
- `ndim` — число измерений;
- `T` — транспонирование.

Основные методы:
- `reshape()`
- `flatten()`
- `astype()`

---
#### Индексация и срезы
Обращение по индексу и срезы проводят также, как и в списках Python:
```
a[0]
a[2:5]
```
Для матриц используются несколько индексов:
```
matrix[1,2]
matrix[:,0]
matrix[:2,:2]
```
Можно получать строки, столбцы и подматрицы.

---
#### Операции над массивами

NumPy поддерживает:
- **Арифметические операции**
	```
	a+b  
	a-b  
	a*b  
	a/b  
	a**b
	```
Они выполняются **поэлементно**.
Если размеры отличаются, используется механизм **broadcasting** (транслирование).

- **Логические операции**
	```
	a>b
	ab
	np.logical_and()
	np.logical_or()
	np.logical_not()
	```
Возвращают массив значений `True` и `False`.

- **Матричные операции**
	умножение;
	транспонирование;
	обращение матрицы.
	
	Также используется модуль `numpy.linalg`:
	```
	np.linalg.inv()
	np.linalg.det()
	```

- **Статистические методы**
	```
	min()
	max()
	sum()
	mean()
	median()
	std()
	argmin()
	argmax()
	```
Их можно применять как ко всему массиву, так и по строкам или столбцам с помощью параметра `axis`.

---
#### Подмодули NumPy
- **numpy.random** — генерация случайных чисел (`np.random.rand`, `np.random.normal`, `np.random.randint`)
- **numpy.linalg** — линейная алгебра (`np.linalg.det`, `np.linalg.inv`)
- **numpy.fft** — преобразование Фурье

#### 4. Обработка табличных данных (pandas, polars)
---
#### Pandas
**Pandas — библиотека Python для анализа и обработки табличных данных.**

Она предоставляет инструменты для:
- хранения данных;
- фильтрации;
- сортировки;
- группировки;
- агрегирования;
- очистки данных;
- объединения таблиц;
- работы с временными рядами.

Также Pandas хорошо интегрируется с NumPy и Matplotlib.

---
#### Основные структуры данных
- ##### **Series** - одномерный массив с индексами.
Пример:
```
import pandas as pd

s = pd.Series([1,2,3,4])
```
Каждый элемент имеет собственный индекс.

- ##### **DataFrame** - основная структура Pandas. 
Представляет собой таблицу, состоящую из строк и столбцов. Каждый столбец является объектом `Series`.

---
#### Основные возможности Pandas
Pandas позволяет:
- читать данные из CSV, Excel, SQL;
- фильтровать строки;
- сортировать таблицы;
- объединять таблицы;
- удалять пропуски (`dropna()`);
- выполнять группировку (`groupby`);
- рассчитывать статистику (`mean()`, `sum()`, `std()`, `median()`);
- подготавливать данные к анализу и машинному обучению.

---
#### Polars
**Polars — современная библиотека для обработки табличных данных.**

Основные особенности:
- высокая скорость работы;
- многопоточность;
- низкое потребление памяти;
- API, похожий на Pandas;
- особенно эффективна при работе с большими наборами данных.

По сравнению с Pandas Polars обычно быстрее на больших объёмах данных благодаря реализации на Rust.

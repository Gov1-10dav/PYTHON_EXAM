## Обработка изображений (pillow)

библиотека pillow - основной инструмент в python для быстрой обработки растровых изображений. она позволяет открывать, изменять, конвертировать формат и сохранять графические файлы

#### использование
```
from PIL import Image, ImageFilter, ImageDraw, ImageFont

# 1. открытие и базовые данные
img = Image.open("img.jpg")
w, h = img.size  # ширина и высота

# 2. изменение размера и обрезка
img_resized = img.resize((800, 600), Image.Resampling.LANCZOS) # качественное сжатие
img_cropped = img.crop((50, 50, 400, 400)) # координаты: x1, y1, x2, y2

# 3. повороты и отражения
img_rotated = img.rotate(90, expand=True) # поворот без обрезки углов
img_flipped = img.transpose(Image.Transpose.FLIP_LEFT_RIGHT) # отражение по горизонтали

# 4. фильтры и цвета
img_gray = img.convert("L") # перевод в черно-белый (оттенки серого)
img_blur = img.filter(ImageFilter.GaussianBlur(radius=5)) # размытие

# 5. рисование и текст
draw = ImageDraw.Draw(img)
draw.rectangle([10, 10, 200, 100], outline="red", width=3) # рамка
# для текста (шрифт arial.ttf должен быть в папке со скриптом или в системе)
font = ImageFont.truetype("arial.ttf", size=24) 
draw.text((20, 20), "Текст", fill="white", font=font)

# 6. сохранение (формат определяется автоматически по расширению)
img.save("result.png")

```

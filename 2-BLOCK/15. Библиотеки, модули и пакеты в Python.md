## 15. Библиотеки, модули и пакеты в Python

 Иерархия организации кода:  **Модули **$\rightarrow$** Пакеты **$\rightarrow$** Библиотеки** .

1. Модули (Modules)Модуль — это один файл .py.
   1. Импорт: import math или from math import pi.
   2. Конструкция if __name__ == "__main__": Позволяет запустить код (тесты, примеры) только тогда, когда файл запущен напрямую как главный скрипт, а не импортирован.
   3. Порядок поиска (sys.path): 1) Директория скрипта  2) Переменная PYTHONPATH  3) Стандартная библиотека  4) Сторонние пакеты (site-packages).
2. Пакеты (Packages)Пакет — это папка, содержащая модули и другие подпакеты.
   1. Файл __init__.py: Нужен для инициализации пакета при импорте. В нем можно прописать список __all__ = [...] для ограничения экспорта при импорте через звездочку (from pkg import *).
3. Библиотеки (Libraries)Библиотека — глобальное понятие, набор пакетов и модулей для конкретных задач.
   1. Стандартная библиотека: Встроена в Python по принципу «батарейки в комплекте» (os, sys, math, datetime, json).
   2. Внешние библиотеки: Устанавливаются из репозитория PyPI через менеджер пакетов pip (requests, numpy, Django).

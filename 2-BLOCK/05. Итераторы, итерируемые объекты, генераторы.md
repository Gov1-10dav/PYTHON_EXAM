### 5. Итераторы, итерируемые объекты, генераторы
---
#### Итерируемый объект
**Итерируемый объект** — любой объект, который умеет **возвращать итератор**. Он реализует метод `__iter__()`, возвращающий итератор. Списки, кортежи, строки, словари, множества — всё это итерируемые объекты (но не итераторы сами по себе).

Проще говоря, **итерируемый объект - это объект, элементы которого можно перебирать в цикле `for`.**
```
a_set = {1, 2, 3}
b_iterator = iter(a_set)  # получаем итератор из итерируемого объекта
```
---
#### Итератор
**Итератор** — объект, реализующий **протокол итератора**: методы `__iter__()` (возвращает `self`) и `__next__()` (возвращает следующий элемент или бросает `StopIteration`).

Проще говоря, **итератор - специальный объект, который возвращает следующий элемент последовательности.**
```
class SimpleIterator:
    def __iter__(self):
        return self

    def __init__(self, limit):
        self.limit = limit
        self.counter = 0

    def __next__(self):
        if self.counter < self.limit:
            self.counter += 1
            return 1
        else:
            raise StopIteration
```
Вручную работать с итератором можно через функции `iter()` и `next()`:
```
lst = [1, 2, 3]
itr = iter(lst)
print(next(itr))  # 1
print(next(itr))  # 2
# При исчерпании элементов - StopIteration
```
---
#### Генераторы
**Генератор** — это более простой способ создания итераторов с помощью ключевого слова `yield`. Функция с `yield` — это **функция-генератор**.
```
def series_generator(low, high):
    while low <= high:
        yield low
        low += 1

for num in series_generator(1, 5):
    print(num)  # 1, 2, 3, 4, 5
```
**Преимущества генераторов:**
- ленивое вычисление;
- экономия памяти;
- возможность обработки больших объёмов данных;
- сохранение состояния между вызовами.

Обычная функция возвращает результат через `return` и завершает работу, а генератор использует `yield` и продолжает выполнение позже.

---
#### Генераторные выражения
Аналог списковых включений:
```
(x*x for x in range(10))
```
Возвращают объект типа `generator`, а не список.


## 13.Кучи, деревья и графы

### 1. Деревья (Trees)

**Дерево** — связный ациклический граф. Имеет один корень, внутренние узлы и листья (узлы без потомков).

* **Бинарное дерево поиска (BST):** Узел имеет **$\le 2$** потомков. Свойство: левый потомок < родитель **$\le$** правый потомок.
  * *Сложность (поиск/вставка/удаление):* в среднем **$\mathcal{O}(\log n)$**, в худшем (вырожденное дерево) **$\mathcal{O}(n)$**. Для балансировки используются AVL или Красно-черные деревья.
* **Обходы дерева:**
  * **DFS (в глубину):** *Прямой* (корень-левый-правый), *Симметричный* (левый-корень-правый — для BST дает сортировку), *Обратный* (левый-правый-корень).
  * **BFS (в ширину):** Поуровневый обход с использованием очереди.

### 2. Кучи (Heaps)

**Куча** — завершенное бинарное дерево, где выполняется  *свойство кучи* :

* **Min-Heap:** корень — минимальный элемент, родитель **$\le$** потомки.
* **Max-Heap:** корень — максимальный элемент, родитель **$\ge$** потомки.
* **Реализация на массиве:** Элемент `i` имеет левого потомка `2*i + 1`, правого `2*i + 2`, родителя `(i-1)//2`.
* **Сложность:** поиск min/max — **$\mathcal{O}(1)$**, вставка/удаление — **$\mathcal{O}(\log n)$**, построение — **$\mathcal{O}(n)$**.
* **В Python:** модуль `heapq` (минимальная куча поверх списка: `heapify(lst)`, `heappush(h, x)`, `heappop(h)`).

### 3. Графы (Graphs)

**Граф** — множество вершин **$V$** и ребер **$E$**: **$G = (V, E)$**. Бывают ориентированные/неориентированные, взвешенные/невзвешенные.

* **Способы представления:**
  * *Матрица смежности:* Таблица **$V \times V$**. Поиск ребера — **$\mathcal{O}(1)$**, память — **$\mathcal{O}(V^2)$** (плохо для разреженных графов).
  * *Список смежности:* Словарь/массив списков соседей. Память — **$\mathcal{O}(V + E)$** (эффективно).
* **Ключевые алгоритмы:**
  * *Обходы:* DFS (стек/рекурсия, поиск циклов), BFS (очередь, кратчайший путь в невзвешенном графе).
  * *Кратчайший путь:* Дейкстра (**$\mathcal{O}(E \log V)$**, веса **$\ge 0$**), Беллман-Форд (любые веса, ищет отр. циклы), Флойд-Уоршелл (**$\mathcal{O}(V^3)$**, между всеми парами).

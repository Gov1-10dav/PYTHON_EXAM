# 12. Использование функций в Python

Функция — это блок кода, который можно вызвать несколько раз.

Функции нужны, чтобы не повторять один и тот же код и делать программу более понятной.

## Определение функции

Для создания функции используется ключевое слово `def`.

```python
def hello():
    print("Привет!")
```

Вызов функции:

```python
hello()
```

Результат:

```python
Привет!
```

## Возврат значения: `return`

`return` возвращает результат работы функции.

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```

Результат:

```python
8
```

## `*args`

`*args` позволяет передать в функцию любое количество обычных аргументов.

Внутри функции они будут храниться как кортеж.

```python
def summa(*args):
    return sum(args)

print(summa(1, 2, 3, 4))
```

Результат:

```python
10
```

## `**kwargs`

`**kwargs` позволяет передать любое количество именованных аргументов.

Внутри функции они будут храниться как словарь.

```python
def info(**kwargs):
    print(kwargs)

info(name="Анна", age=18)
```

Результат:

```python
{'name': 'Анна', 'age': 18}
```

## `lambda`

`lambda` — это короткая функция в одну строку без имени.

```python
add = lambda a, b: a + b

print(add(2, 3))
```

Результат:

```python
5
```

Обычно `lambda` используют для простых действий.

## Главное

Функции помогают разделять программу на понятные части.  
`def` создает функцию, `return` возвращает результат, `*args` принимает много обычных аргументов, а `**kwargs` принимает много именованных аргументов.

# 3. Характеристика модулей обработки чисел в Python

В Python есть несколько встроенных модулей для работы с числами. Они нужны, чтобы выполнять математические операции, генерировать случайные числа, точно считать десятичные дроби, работать с обыкновенными дробями и получать простую статистику по данным.

Коротко:

| Модуль | Для чего нужен |
|---|---|
| `math` | обычная математика: корни, степени, тригонометрия, округление, факториал |
| `random` | генерация случайных чисел и случайный выбор элементов |
| `decimal` | точные десятичные вычисления, особенно для денег |
| `fractions` | работа с обыкновенными дробями |
| `statistics` | среднее, медиана, мода, дисперсия, стандартное отклонение |
| `cmath` | работа с комплексными числами |

---

## 1. Числовые типы в Python

Перед модулями важно помнить основные числовые типы:

| Тип | Что хранит | Пример |
|---|---|---|
| `int` | целые числа | `10`, `-5`, `1000` |
| `float` | вещественные числа | `3.14`, `0.5` |
| `complex` | комплексные числа | `2 + 3j` |

Пример:

```python
x = 10          # int
y = 3.14        # float
z = 2 + 3j      # complex
```

У типа `float` есть особенность: он хранит числа приближённо. Поэтому иногда получаются неожиданные результаты:

```python
print(0.1 + 0.2)
# 0.30000000000000004
```

Это не ошибка Python, а особенность хранения вещественных чисел в памяти компьютера. Если нужна точность, используют `decimal`.

---

## 2. Модуль `math`

`math` — это стандартный модуль для математических операций.

Подключение:

```python
import math
```

Основные функции:

| Функция | Что делает |
|---|---|
| `math.sqrt(x)` | квадратный корень |
| `math.factorial(x)` | факториал |
| `math.floor(x)` | округление вниз |
| `math.ceil(x)` | округление вверх |
| `math.trunc(x)` | отбрасывает дробную часть |
| `math.gcd(a, b)` | наибольший общий делитель |
| `math.lcm(a, b)` | наименьшее общее кратное |
| `math.sin(x)` | синус |
| `math.cos(x)` | косинус |
| `math.tan(x)` | тангенс |
| `math.radians(x)` | перевод градусов в радианы |
| `math.degrees(x)` | перевод радиан в градусы |

Пример:

```python
import math

print(math.sqrt(25))       # 5.0
print(math.factorial(5))   # 120
print(math.floor(3.9))     # 3
print(math.ceil(3.1))      # 4
print(math.gcd(12, 18))    # 6
```

В `math` также есть математические константы:

```python
import math

print(math.pi)  # число π
print(math.e)   # число e
```

### Важно про тригонометрию

Функции `sin`, `cos`, `tan` работают с радианами, а не с градусами.

```python
import math

angle = 90
radians = math.radians(angle)

print(math.sin(radians))
# 1.0
```

## 3. Модуль `random`

`random` нужен для генерации случайных чисел и случайного выбора элементов.

Подключение:

```python
import random
```

Основные функции:

| Функция | Что делает |
|---|---|
| `random.random()` | случайное число от `0` до `1` |
| `random.randint(a, b)` | случайное целое число от `a` до `b` включительно |
| `random.randrange(start, stop, step)` | случайное число из диапазона |
| `random.choice(list)` | случайный элемент из списка |
| `random.shuffle(list)` | перемешивает список |
| `random.sample(list, k)` | выбирает `k` случайных элементов без повторений |
| `random.seed(x)` | фиксирует генерацию случайных чисел |

Примеры:

```python
import random

print(random.random())        # например 0.5738
print(random.randint(1, 10))  # число от 1 до 10
```

Случайный выбор:

```python
import random

names = ["Аня", "Петя", "Иван"]
print(random.choice(names))
```

Перемешивание списка:

```python
import random

numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print(numbers)
```

`random.seed()` нужен, чтобы случайные числа повторялись при каждом запуске. Это удобно для тестов и экспериментов.

```python
import random

random.seed(10)
print(random.randint(1, 100))
```

На экзамене можно сказать: `random` используют в играх, тестировании, моделировании и случайной выборке данных.

## 4. Модуль `decimal`

`decimal` нужен для точных десятичных вычислений.

Обычный `float` может давать неточность:

```python
print(0.1 + 0.2)
# 0.30000000000000004
```

С `Decimal` можно считать точнее:

```python
from decimal import Decimal

x = Decimal("0.1")
y = Decimal("0.2")

print(x + y)
# 0.3
```

Важно: `Decimal` лучше создавать из строки, а не из `float`.

Правильно:

```python
Decimal("0.1")
```

Нежелательно:

```python
Decimal(0.1)
```

### Округление через `quantize()`

`quantize()` позволяет округлить число до нужного количества знаков.

```python
from decimal import Decimal

price = Decimal("10.567")
result = price.quantize(Decimal("0.01"))

print(result)
# 10.57
```

`decimal` часто используют там, где важна точность: деньги, финансы, банковские операции, расчёты цен.

## 5. Модуль `fractions`

`fractions` нужен для работы с обыкновенными дробями.

Главный класс — `Fraction`.

```python
from fractions import Fraction

x = Fraction(1, 3)
y = Fraction(1, 6)

print(x + y)
# 1/2
```

Python сам сокращает дроби:

```python
from fractions import Fraction

print(Fraction(2, 4))
# 1/2
```

Можно преобразовать десятичное число в дробь:

```python
from fractions import Fraction

print(Fraction("0.25"))
# 1/4
```

Метод `limit_denominator()` подбирает простую дробь, близкую к числу:

```python
from fractions import Fraction

x = Fraction(3.14159).limit_denominator(100)
print(x)
```

`fractions` полезен, когда надо получить точный результат в виде дроби, а не десятичного приближения.

## 6. Модуль `statistics`

`statistics` нужен для простого статистического анализа числовых данных.

Подключение:

```python
import statistics
```

Основные функции:

| Функция | Что делает |
|---|---|
| `statistics.mean(data)` | среднее арифметическое |
| `statistics.median(data)` | медиана |
| `statistics.mode(data)` | мода, самое частое значение |
| `statistics.variance(data)` | дисперсия |
| `statistics.stdev(data)` | стандартное отклонение |

Пример:

```python
import statistics

data = [4, 5, 5, 3, 4, 5]

print(statistics.mean(data))    # среднее
print(statistics.median(data))  # медиана
print(statistics.mode(data))    # самое частое значение
```

### Среднее, медиана и мода простыми словами

- **Среднее** — сумма всех значений, делённая на их количество.
- **Медиана** — центральное значение в отсортированном наборе.
- **Мода** — значение, которое встречается чаще всего.

Пример:

```python
data = [1, 2, 100]
```

Среднее будет большим из-за `100`, а медиана будет `2`. Поэтому медиана часто устойчивее к выбросам.

## 7. Модуль `cmath`

`cmath` похож на `math`, но используется для комплексных чисел.

Комплексное число в Python записывается через `j`:

```python
z = 2 + 3j
```

Пример:

```python
import cmath

z = 4 + 0j
print(cmath.sqrt(z))
```

`cmath` используют реже, но он нужен, если в задаче есть комплексные числа.

Модуль `statistics` нужен для простого статистического анализа: среднего, медианы, моды, дисперсии и стандартного отклонения. Также есть `cmath` для комплексных чисел.**

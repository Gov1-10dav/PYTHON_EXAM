# 6. Обработка списков в Python

`list` — это изменяемая упорядоченная коллекция произвольных объектов.

В списке можно хранить числа, строки, другие списки и любые другие объекты.

```python
numbers = [1, 2, 3, 4]
names = ["Анна", "Иван", "Петр"]
```

## Основные методы списков

### `append()`

Добавляет один элемент в конец списка.

```python
numbers = [1, 2, 3]
numbers.append(4)
print(numbers)
```

Результат:

```python
[1, 2, 3, 4]
```

---

### `extend()`

Добавляет несколько элементов в конец списка.

```python
numbers = [1, 2]
numbers.extend([3, 4, 5])
print(numbers)
```

Результат:

```python
[1, 2, 3, 4, 5]
```

---

### `insert()`

Вставляет элемент по указанному индексу.

```python
numbers = [1, 3, 4]
numbers.insert(1, 2)
print(numbers)
```

Результат:

```python
[1, 2, 3, 4]
```

---

### `remove()`

Удаляет элемент по значению.

```python
numbers = [1, 2, 3, 2]
numbers.remove(2)
print(numbers)
```

Результат:

```python
[1, 3, 2]
```

Удаляется только первое найденное значение.

---

### `pop()`

Удаляет элемент по индексу и возвращает его.

```python
numbers = [10, 20, 30]
x = numbers.pop(1)

print(x)
print(numbers)
```

Результат:

```python
20
[10, 30]
```

Если индекс не указан, удаляется последний элемент.

---

### `sort()`

Сортирует список.

```python
numbers = [3, 1, 2]
numbers.sort()
print(numbers)
```

Результат:

```python
[1, 2, 3]
```

---

### `reverse()`

Разворачивает список в обратном порядке.

```python
numbers = [1, 2, 3]
numbers.reverse()
print(numbers)
```

Результат:

```python
[3, 2, 1]
```

## Главное

Списки изменяемые, поэтому их можно дополнять, удалять из них элементы, сортировать и менять порядок элементов.

# 9. Обработка словарей в Python

`dict` — это коллекция пар «ключ-значение».

Каждый ключ уникален и связан с определенным значением.

```python
student = {
    "name": "Иван",
    "age": 18,
    "group": "ИС-101"
}
```

## Создание словаря

Через фигурные скобки:

```python
person = {"name": "Анна", "age": 20}
```

Через функцию `dict()`:

```python
person = dict(name="Анна", age=20)
```

## Доступ к значениям

### Через квадратные скобки `[]`

```python
student = {"name": "Иван", "age": 18}

print(student["name"])
```

Результат:

```python
Иван
```

Если ключа нет, будет ошибка.

```python
print(student["city"])
```

Ошибка:

```python
KeyError
```

---

### Через `get()`

Метод `get()` безопасно получает значение по ключу.

```python
student = {"name": "Иван", "age": 18}

print(student.get("city"))
```

Результат:

```python
None
```

Можно указать значение по умолчанию:

```python
print(student.get("city", "Город не указан"))
```

Результат:

```python
Город не указан
```

## Методы словарей

### `keys()`

Возвращает все ключи.

```python
student = {"name": "Иван", "age": 18}

print(student.keys())
```

---

### `values()`

Возвращает все значения.

```python
print(student.values())
```

---

### `items()`

Возвращает пары ключ-значение.

```python
print(student.items())
```

---

### `update()`

Обновляет словарь или добавляет новые пары.

```python
student.update({"city": "Москва"})
print(student)
```

---

### `pop()`

Удаляет значение по ключу и возвращает его.

```python
age = student.pop("age")

print(age)
print(student)
```

## Главное

Словарь удобен, когда нужно хранить данные в формате «название — значение», например имя, возраст, группа, оценка.

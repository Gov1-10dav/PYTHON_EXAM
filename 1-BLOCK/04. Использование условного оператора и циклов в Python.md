# 4. Использование условного оператора и циклов в Python

Условный оператор нужен, чтобы программа выполняла разные действия в зависимости от условия.

## `if`

`if` проверяет условие.

```python
age = 18

if age >= 18:
    print("Доступ разрешен")
```

Если условие истинное, код внутри `if` выполнится.

## `else`

`else` выполняется, если условие в `if` ложное.

```python
age = 16

if age >= 18:
    print("Доступ разрешен")
else:
    print("Доступ запрещен")
```

## `elif`

`elif` используется для проверки дополнительных условий.

```python
mark = 4

if mark == 5:
    print("Отлично")
elif mark == 4:
    print("Хорошо")
elif mark == 3:
    print("Удовлетворительно")
else:
    print("Плохо")
```

## Операторы сравнения

| Оператор | Значение |
|---|---|
| `==` | равно |
| `!=` | не равно |
| `>` | больше |
| `<` | меньше |
| `>=` | больше или равно |
| `<=` | меньше или равно |

## Логические операторы

- `and` — и;
- `or` — или;
- `not` — не.

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("Можно войти")
```

## Главное

`if-elif-else` — это конструкция ветвления.  
Она позволяет выполнять разные блоки кода в зависимости от условий.

# 4. Использование условного оператора и циклов в Python

Условный оператор нужен, чтобы программа выполняла разные действия в зависимости от условия.

## `if`

`if` проверяет условие.

```python
age = 18

if age >= 18:
    print("Доступ разрешен")
```

Если условие истинное, код внутри `if` выполнится.

## `else`

`else` выполняется, если условие в `if` ложное.

```python
age = 16

if age >= 18:
    print("Доступ разрешен")
else:
    print("Доступ запрещен")
```

## `elif`

`elif` используется для проверки дополнительных условий.

```python
mark = 4

if mark == 5:
    print("Отлично")
elif mark == 4:
    print("Хорошо")
elif mark == 3:
    print("Удовлетворительно")
else:
    print("Плохо")
```

## Операторы сравнения

| Оператор | Значение |
|---|---|
| `==` | равно |
| `!=` | не равно |
| `>` | больше |
| `<` | меньше |
| `>=` | больше или равно |
| `<=` | меньше или равно |

## Логические операторы

- `and` — и;
- `or` — или;
- `not` — не.

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("Можно войти")
```

## Главное

`if-elif-else` — это конструкция ветвления.  
Она позволяет выполнять разные блоки кода в зависимости от условий.

# Циклы в Python

Циклы нужны, чтобы выполнять один и тот же блок кода несколько раз. В Python чаще всего используются два цикла: `for` и `while`.

## Цикл `for`

`for` используется, когда нужно перебрать элементы последовательности: список, строку, кортеж, словарь или диапазон чисел.

```python
names = ["Анна", "Иван", "Олег"]

for name in names:
    print(name)
```

В этом примере переменная `name` по очереди принимает значения из списка `names`.

Часто `for` используют вместе с `range()`:

```python
for i in range(5):
    print(i)
```

`range(5)` создаёт числа от `0` до `4`. То есть верхняя граница не включается.

Можно указать начало, конец и шаг:

```python
for i in range(1, 10, 2):
    print(i)
```

Результат: `1, 3, 5, 7, 9`.

## Цикл `while`

`while` выполняет блок кода, пока условие истинно.

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

Здесь цикл работает, пока `count < 5`. Важно менять переменную внутри цикла, иначе можно получить бесконечный цикл.

## `break`

`break` полностью останавливает цикл.

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

Когда `i` станет равен `5`, цикл завершится.

## `continue`

`continue` пропускает текущую итерацию и переходит к следующей.

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

Число `2` не выведется, потому что на этой итерации сработает `continue`.

## `pass`

`pass` ничего не делает. Он нужен как заглушка, когда синтаксически блок кода должен быть, но логика пока не написана.

```python
if age >= 18:
    pass
```

## Вложенные циклы

Вложенный цикл — это цикл внутри другого цикла. Например, так можно пройти по таблице или матрице.

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6]
]

for row in matrix:
    for item in row:
        print(item)
```

Внешний цикл перебирает строки, а внутренний — элементы внутри каждой строки.

## `for` или `while` — что выбрать?

`for` удобен, когда заранее известно, по чему нужно пройти: список, строка, диапазон чисел.

`while` удобен, когда количество повторений заранее неизвестно, и цикл зависит от условия.

Например, `for` — пройти по списку студентов, а `while` — выполнять ввод, пока пользователь не введёт правильное значение.

## Короткий ответ на экзамене

Условный оператор `if-elif-else` используется для ветвления программы: разные блоки кода выполняются в зависимости от условия. Условия строятся с помощью операторов сравнения и логических операторов `and`, `or`, `not`.

Циклы нужны для многократного выполнения кода. Цикл `for` используется для перебора элементов последовательности или диапазона `range()`. Цикл `while` выполняется, пока условие истинно. Оператор `break` полностью завершает цикл, `continue` пропускает текущую итерацию, а `pass` используется как пустая заглушка. Главное в циклах — правильно задавать условие и соблюдать отступы.

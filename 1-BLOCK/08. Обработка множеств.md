# 8. Обработка множеств в Python

`set` — это неупорядоченная коллекция уникальных элементов.

В множестве не может быть одинаковых элементов.

```python
numbers = {1, 2, 3, 3, 4}
print(numbers)
```

Результат:

```python
{1, 2, 3, 4}
```

## Создание множества

Множество можно создать через фигурные скобки:

```python
a = {1, 2, 3}
```

Или через функцию `set()`:

```python
b = set([1, 2, 2, 3])
```

## Операции над множествами

### Объединение: `|`

Объединение возвращает все элементы из обоих множеств.

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)
```

Результат:

```python
{1, 2, 3, 4, 5}
```

---

### Пересечение: `&`

Пересечение возвращает только общие элементы.

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a & b)
```

Результат:

```python
{3}
```

---

### Разность: `-`

Разность возвращает элементы из первого множества, которых нет во втором.

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a - b)
```

Результат:

```python
{1, 2}
```

---

### Симметричная разность: `^`

Симметричная разность возвращает элементы, которые есть только в одном из множеств.

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a ^ b)
```

Результат:

```python
{1, 2, 4, 5}
```

## Главное

Множества нужны, когда важны только уникальные элементы и не важен порядок.

# 10. Обработка даты и времени в Python

В Python для работы с датой и временем используют модули `datetime`, `time`, `calendar`.

## `datetime`

Главный модуль для даты и времени.

```python
from datetime import datetime

now = datetime.now()
print(now)
```

`datetime.now()` возвращает текущую дату и время.

## Создание даты

```python
from datetime import datetime

date = datetime(2026, 6, 22, 12, 30)
print(date)
```

Здесь указываются: год, месяц, день, часы, минуты.

## Получение частей даты

```python
print(now.year)
print(now.month)
print(now.day)
print(now.hour)
print(now.minute)
```

Так можно получить отдельно год, месяц, день, часы и минуты.

## `strftime()`

Преобразует дату в строку.

```python
print(now.strftime("%d.%m.%Y"))
```

Результат:

```python
22.06.2026
```

## `strptime()`

Преобразует строку в дату.

```python
date = datetime.strptime("22.06.2026", "%d.%m.%Y")
print(date)
```

## `timedelta`

Используется для прибавления или вычитания времени.

```python
from datetime import timedelta

tomorrow = now + timedelta(days=1)
print(tomorrow)
```

## `time`

Модуль `time` работает со временем в секундах.

```python
import time

time.sleep(2)
```

`time.sleep(2)` делает паузу на 2 секунды.

## `calendar`

Модуль `calendar` работает с календарем.

```python
import calendar

print(calendar.month(2026, 6))
print(calendar.isleap(2024))
```

## Главное

- `datetime` — дата и время;
- `strftime()` — дату в строку;
- `strptime()` — строку в дату;
- `timedelta` — прибавить или вычесть время;
- `time.sleep()` — пауза;
- `calendar` — календарь.

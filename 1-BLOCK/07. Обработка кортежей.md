# 7. Обработка кортежей в Python

`tuple` — это неизменяемая упорядоченная коллекция.

Кортеж похож на список, но его нельзя изменить после создания.

```python
numbers = (1, 2, 3)
```

## Особенности кортежей

Кортежи:

- хранят элементы в определенном порядке;
- могут содержать разные типы данных;
- не изменяются после создания;
- работают быстрее списков;
- используются для данных, которые не должны меняться.

Пример:

```python
person = ("Иван", 18, "Москва")
```

## Создание кортежа

Обычно кортеж создается через круглые скобки:

```python
data = (10, 20, 30)
```

Кортеж из одного элемента обязательно создается с запятой:

```python
one = (5,)
```

Без запятой это будет обычное число:

```python
one = (5)
```

## Методы кортежей

### `count()`

Считает, сколько раз элемент встречается в кортеже.

```python
numbers = (1, 2, 2, 3, 2)

print(numbers.count(2))
```

Результат:

```python
3
```

---

### `index()`

Возвращает индекс первого найденного элемента.

```python
numbers = (10, 20, 30)

print(numbers.index(20))
```

Результат:

```python
1
```

## Главное

Кортежи используют тогда, когда данные должны храниться в порядке, но не должны изменяться.

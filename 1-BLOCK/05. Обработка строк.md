# 5. Обработка строк в Python

`str` — это неизменяемая последовательность символов.

Строки используются для хранения и обработки текста: слов, предложений, путей к файлам, данных от пользователя и т.д.

## Основные методы строк

### `split()`

Разделяет строку на список по указанному разделителю.

```python
text = "Python это просто"
words = text.split()
print(words)
```

Результат:

```python
['Python', 'это', 'просто']
```

---

### `join()`

Соединяет элементы списка в одну строку.

```python
words = ["Python", "это", "просто"]
text = " ".join(words)
print(text)
```

Результат:

```python
Python это просто
```

---

### `replace()`

Заменяет одну подстроку на другую.

```python
text = "Я изучаю Java"
new_text = text.replace("Java", "Python")
print(new_text)
```

Результат:

```python
Я изучаю Python
```

---

### `strip()`

Удаляет пробелы, переносы строк и символы табуляции в начале и конце строки.

```python
text = "   Python   "
print(text.strip())
```

Результат:

```python
Python
```

---

### `find()`

Ищет подстроку и возвращает индекс первого вхождения.

Если подстрока не найдена, возвращает `-1`.

```python
text = "Я изучаю Python"
print(text.find("Python"))
```

Результат:

```python
9
```

---

### `format()` и f-strings

Используются для форматирования строк.

```python
name = "Алексей"
age = 18

print("Меня зовут {}, мне {} лет".format(name, age))
print(f"Меня зовут {name}, мне {age} лет")
```

Результат:

```python
Меня зовут Алексей, мне 18 лет
```

## Главное

Строки в Python нельзя изменить напрямую, потому что они неизменяемые. Все методы строк возвращают новую строку.

### 2. **Типы данных в программировании**
---
#### **Тип данных** — это множество значений и набор операций, которые можно выполнять над этими значениями. В Python переменная — это лишь ссылка на объект определенного типа в памяти.

---
***Все типы данных в Python можно разделить на несколько основных групп:***
#### 1. Числовые типы (Неизменяемые)

- **`int` (Целые числа)
- **`float` (Вещественные числа / числа с плавающей точкой)**
- **`complex` (Комплексные числа)**
- **`decimal.Decimal` (Точные десятичные дроби)**
- **`fractions.Fraction` (Рациональные дроби)**
---
#### 2. Логический тип (Неизменяемый)

- **`bool` (Булев тип):** 
	Может принимать только два значения: `True` (Истина) или `False` (Ложь). Используется в условных операторах и циклах.
---
#### 3. Последовательности (Sequence Types)

- **`str` (Строки):** 
	Неизменяемая последовательность Unicode-символов. Поддерживают конкатенацию (`+`), умножение на число (`*`), срезы (`[x:y]`), проверку вхождения (`in`).

- **`list` (Списки):** 
	**Изменяемая**, **упорядоченная** последовательность. Может содержать элементы разных типов. Основные методы: `append()`, `extend()`, `sort()`, `pop()`, `del`.

- **`tuple` (Кортежи):** 
	**Неизменяемая**, **упорядоченная** последовательность. Работает быстрее списков, занимает меньше памяти. Используется для безопасного хранения данных и возврата нескольких значений из функций.
---
#### 4. Множества (Set Types)

- **`set` (Множество):** 
	**Изменяемая**, **неупорядоченная коллекция** **уникальных** элементов. Поддерживает математические операции: объединение (`|`), пересечение (`&`), разность (`-`).

- **`frozenset` (Неизменяемое множество):** 
	Может выступать в качестве ключа словаря или элемента другого множества.
---
#### 5. Словари (Mapping Type)

- **`dict` (Словарь):** 
	**Изменяемая коллекция** пар «ключ-значение». Реализована на основе **хеш-таблиц**, что обеспечивает молниеносный поиск по ключу. Ключом может быть только неизменяемый тип данных (число, строка, кортеж).
---
#### 6. Специальный тип

- **`NoneType` (`None`):**
	Обозначает отсутствие значения (пустоту).

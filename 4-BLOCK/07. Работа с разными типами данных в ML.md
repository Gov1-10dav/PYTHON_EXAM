## Типы признаков (Features)

| Тип | Описание | Пример |
|-----|---------|--------|
| **Вещественные** | Любое число на числовой прямой | Температура, рост, цена |
| **Бинарные** | Одно из двух значений | Пол (М/Ж), болен/здоров |
| **Категориальные** | Ограниченное множество, без порядка | Город, профессия |
| **Порядковые (ранги)** | Ограниченное множество с порядком | Рейтинг, оценка |
| **Множественные** | Несколько значений одновременно | Интересы, теги |

## Работа с числовыми признаками

### Масштабирование (Scaling)

Когда признаки имеют разный масштаб, модели (особенно градиентный спуск) работают хуже. Решение — привести признаки к одному порядку.

```python
from sklearn.preprocessing import StandardScaler, MinMaxScaler
import numpy as np

X = np.array([[1, 1000], [2, 2000], [3, 1500]])

# StandardScaler: среднее = 0, std = 1 (диапазон ≈ [-2, 2])
scaler_std = StandardScaler()
X_std = scaler_std.fit_transform(X)

# MinMaxScaler: диапазон [0, 1]
scaler_mm = MinMaxScaler()
X_mm = scaler_mm.fit_transform(X)
```

**StandardScaler** — если данные нормально распределены.  
**MinMaxScaler** — если нужен строгий диапазон [0, 1].

Масштабирование важно для: линейной регрессии, SVM, KNN, нейросетей. Деревья и градиентный бустинг устойчивы к масштабу.

## Работа с категориальными признаками

### One-Hot Encoding (OHE)

Создаёт бинарный столбец для каждой категории.

```python
import pandas as pd
from sklearn.preprocessing import OneHotEncoder

df = pd.DataFrame({'city': ['Moscow', 'SPb', 'Moscow', 'Kazan']})

encoder = OneHotEncoder(sparse=False, drop='first')  # drop='first' — убираем мультиколлинеарность
encoded = encoder.fit_transform(df[['city']])
print(encoded)
```

Или через pandas:
```python
dummies = pd.get_dummies(df['city'], drop_first=True)
```

Минус: число столбцов = число категорий (при большом числе категорий — «проклятие размерности»).

### Label Encoding

Заменяет категории числами (подходит для порядковых признаков).

```python
from sklearn.preprocessing import LabelEncoder

le = LabelEncoder()
df['city_encoded'] = le.fit_transform(df['city'])
```

⚠️ Нельзя применять к номинальным категориям (модель воспримет числа как порядок).

### Target Encoding (счётчики)

Заменяет категорию средним значением таргета по этой категории.

```python
# Вычисление счётчиков вручную
target_means = df.groupby('city')['target'].mean()
df['city_encoded'] = df['city'].map(target_means)
```

Риск переобучения — используют вместе с кросс-валидацией.

## Работа с пропущенными значениями (Missing Values)

Удалять строки с пропусками нежелательно — теряется информация для обучения.

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'price': [100, np.nan, 200, np.nan, 150],
    'category': ['A', 'B', np.nan, 'A', 'B']
})

# Заполнение средним
mean_val = df['price'].mean()
df['price'].fillna(mean_val, inplace=True)

# Заполнение медианой
median_val = df['price'].median()

# Заполнение самой популярной категорией
popular = df['category'].value_counts().index[0]
df['category'].fillna(popular, inplace=True)

# Заполнение новой категорией «Неизвестно»
df['category'].fillna('Unknown', inplace=True)

# Заполнение средним по группе
grouped_mean = df.groupby('category')['price'].transform('mean')
df['price'].fillna(grouped_mean, inplace=True)
```

**Когда что применять:**
- Числа: среднее, медиана, значение по группе.
- Категории: самый частый класс или отдельная метка «Unknown».
- Временные ряды: предыдущее/следующее значение (`ffill`/`bfill`).

## Работа с выбросами (Outliers)

**Выброс** — точка, резко отличающаяся от остальных. Стягивает линию регрессии на себя, увеличивая ошибку.

### Обнаружение через IQR (Boxplot)

```python
import pandas as pd

Q1 = df['price'].quantile(0.25)
Q3 = df['price'].quantile(0.75)
IQR = Q3 - Q1

lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

# Отметить выбросы
outliers = df[(df['price'] < lower) | (df['price'] > upper)]

# Удалить выбросы
df_clean = df[(df['price'] >= lower) & (df['price'] <= upper)]
```

### Обнаружение через Z-score

```python
from scipy import stats

z_scores = abs(stats.zscore(df['price']))
df_clean = df[z_scores < 3]  # убрать значения дальше 3σ
```

### Что делать с выбросами?
- Удалить строку.
- Заменить на границу (winsorizing).
- Использовать устойчивые к выбросам алгоритмы (медиана, MAE, деревья решений).

## Отбор признаков (Feature Selection)

### EDA (Exploratory Data Analysis)

```python
import matplotlib.pyplot as plt
import seaborn as sns

# ScatterPlot — корреляция признака с таргетом
sns.scatterplot(x='feature', y='target', data=df)

# Barplot — категориальный признак vs таргет
df.groupby('category')['target'].mean().plot.bar()

# Boxplot — распределение таргета по категориям
df.boxplot(column='target', by='category')

# Корреляционная матрица
corr_matrix = df.corr()
sns.heatmap(corr_matrix, annot=True)
```

### Статистические методы (фильтрация)

```python
# Дисперсия — убрать константные признаки
from sklearn.feature_selection import VarianceThreshold

selector = VarianceThreshold(threshold=0.0)
X_reduced = selector.fit_transform(X)

# Корреляция с таргетом
correlations = df.corr()['target'].abs().sort_values(ascending=False)
```

### Встроенные методы (Lasso, Random Forest)

```python
from sklearn.linear_model import Lasso

lasso = Lasso(alpha=0.1)
lasso.fit(X_train, y_train)
# Нулевые коэффициенты → признак не важен
important = X.columns[lasso.coef_ != 0]
```

## Понижение размерности (PCA)

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)

print(pca.explained_variance_ratio_)  # доля объяснённой дисперсии
```

Используется когда признаков слишком много и они коррелируют между собой.

## Итоговая шпаргалка

| Задача | Инструмент |
|--------|-----------|
| Числа → одинаковый масштаб | `StandardScaler`, `MinMaxScaler` |
| Категории → числа (без порядка) | `OneHotEncoder`, `get_dummies` |
| Категории → числа (с порядком) | `LabelEncoder`, `OrdinalEncoder` |
| Категории → таргет | Target Encoding (с кросс-валидацией) |
| Пропуски | `fillna(mean/median/mode)` |
| Выбросы | IQR, Z-score, удаление/замена |
| Отбор признаков | EDA, Lasso, Random Forest, фильтрация |
| Снижение размерности | PCA |

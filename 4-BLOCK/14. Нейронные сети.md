### 2. Процесс обучения нейросети

Обучение состоит из двух чередующихся шагов:

1. **Прямое распространение (Forward Propagation):** Данные проходят через входной слой, скрытые слои к выходному слою, где вычисляется предсказание $\hat{y}$. На основе $\hat{y}$ и реального ответа $y$ считается значение **функции потерь** (Loss function: например, MSE для регрессии или Cross-Entropy для классификации).
2. **Обратное распространение ошибки (Backpropagation):** Алгоритм вычисления градиентов функции потерь по всем весам сети с помощью цепного правила дифференцирования (*Chain Rule*). Ошибка распространяется от выходного слоя обратно к входному.

* **SGD (Стохастический градиентный спуск):** Обновляет веса по мини-батчам данных.
* **Adam (Adaptive Moment Estimation):** Адаптивный оптимизатор, который подбирает индивидуальный шаг обучения (learning rate) для каждого веса, сочетая идеи накопления импульса (Momentum) и среднеквадратичного градиента (RMSProp).

### 3. Переобучение (Overfitting) и регуляризация

Нейросети имеют миллионы параметров и склонны запоминать обучающую выборку. Способы борьбы:

* **Dropout (Исключение):** На каждом шаге обучения случайным образом "выключается" заданный процент нейронов. Сеть учится быть устойчивой и не полагаться на конкретные связи.
* **L1 / L2 регуляризация:** Добавление штрафа за слишком большие значения весов в функцию потерь.
* **Batch Normalization:** Нормализация выходов слоев внутри каждого батча. Стабилизирует и ускоряет обучение.
* **Early Stopping (Ранняя остановка):** Прекращение обучения, когда ошибка на валидационной выборке начинает расти, несмотря на падение ошибки на тренировочной.

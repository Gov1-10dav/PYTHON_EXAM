**Градиентный бустинг (Gradient Boosting)** - это ансамблевый метод машинного обучения, который последовательно строит слабые модели (обычно деревья решений) таким образом, чтобы каждая следующая модель исправляла ошибки предыдущих.

**Основные принципы:**

Бустинг строит модель в виде суммы слабых моделей:
F(x) = F₀(x) + α₁·h₁(x) + α₂·h₂(x) + ... + α_M·h_M(x)

Где:
- F₀(x) - начальное приближение (константа)
- h_i(x) - слабые модели (обычно деревья)
- α_i - веса (коэффициенты обучения)

**Обучение градиентного бустинга:**
1. Начинается с начальной модели (например, среднее значение для регрессии)
2. На каждой итерации вычисляются градиенты функции потерь для текущей модели
3. Обучается новая слабая модель, которая аппроксимирует эти градиенты (которые являются отрицательными)
4. Модель добавляется с учетом коэффициента обучения
5. Процесс повторяется заданное количество раз

**Основные реализации:**

**XGBoost (Extreme Gradient Boosting)** - одна из самых популярных реализаций:
- Использует вторые производные (гессиан) для более точной оптимизации
- Регуляризация для борьбы с переобучением
- Эффективная работа с пропущенными значениями
- Поддержка параллелизации и кэширования

**LightGBM**:
- Использует построение деревьев по листьям (leaf-wise), а не по уровням
- Более быстрый и эффективный для больших данных
- Поддерживает категориальные признаки напрямую

**CatBoost**:
- Специально оптимизирован для работы с категориальными признаками
- Устойчив к переобучению
- Хорошо работает с небольшими датасетами

**Преимущества градиентного бустинга:**
- Высокая точность предсказаний
- Эффективность на структурированных данных
- Возможность работы с различными функциями потерь
- Хорошо интерпретируемая важность признаков

**Недостатки:**
- Чувствительность к настройке гиперпараметров
- Склонность к переобучению при неправильной настройке
- Меньшая интерпретируемость по сравнению с одиночным деревом
- Требовательность к вычислительным ресурсам

**Гиперпараметры градиентного бустинга:**
- Количество деревьев (n_estimators)
- Скорость обучения (learning_rate) - множитель для весов деревьев
- Глубина деревьев (max_depth)
- Минимальное количество образцов в листе
- Доля подвыборки для обучения каждого дерева (subsample)
- Регуляризация (alpha, lambda)

**Области применения:** соревнования по машинному обучению, задачи классификации и регрессии с табличными данными, кредитный скоринг, прогнозирование временных рядов, рекомендательные системы.

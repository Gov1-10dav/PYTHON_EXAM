# 8. Модель линейной регрессии

---

## Что такое линейная регрессия?

**Линейная регрессия** — задача машинного обучения с учителем, направленная на предсказание **вещественного числового значения** (таргета) на основе признаков.

Уравнение модели:

```
a(x) = β₀ + β₁·d₁ + β₂·d₂ + ... + βₙ·dₙ
```

где `d₁...dₙ` — признаки объекта, `β` — веса (коэффициенты модели).

---

## Метод наименьших квадратов (МНК / OLS)

Цель — найти коэффициенты β, при которых сумма квадратов отклонений минимальна:

```
Q(β) = (1/m) Σ (a(xᵢ) − yᵢ)²  →  min
```

**Матричная форма:**
```
β* = (XᵀX)⁻¹ · Xᵀ · Y
```

Ограничения: матрица XᵀX должна быть обратимой (нет мультиколлинеарности).

---

## Метрики качества регрессии

| Метрика | Формула | Особенности |
|---------|---------|-------------|
| **MSE** | (1/n)Σ(yᵢ−ŷᵢ)² | Штрафует крупные ошибки сильнее |
| **RMSE** | √MSE | Тот же масштаб, что у таргета |
| **MAE** | (1/n)Σ|yᵢ−ŷᵢ| | Устойчив к выбросам |
| **R²** | 1 − SSres/SStot | Безразмерный; R²=1 → идеал, R²<0 → хуже среднего |

---

## Реализация через scipy.stats

```python
import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt

x = np.array([2.2, 2.8, 3.0, 3.5, 3.9, 4.2, 4.4, 4.8, 5.1, 5.5])
y = np.array([0.13, 0.21, 0.28, 0.26, 0.23, 0.29, 0.28, 0.24, 0.31, 0.33])

# Линейная регрессия
res = stats.linregress(x, y)

print(f"slope     = {res.slope:.4f}")        # b₁
print(f"intercept = {res.intercept:.4f}")    # b₀
print(f"R         = {res.rvalue:.4f}")       # коэф. корреляции
print(f"R²        = {res.rvalue**2:.4f}")    # коэф. детерминации
print(f"p-value   = {res.pvalue:.6f}")       # значимость наклона
print(f"std_err   = {res.stderr:.4f}")       # ошибка slope

print(f"y = {res.intercept:.3f} + {res.slope:.3f} * x")

# Прогноз
y_pred = res.intercept + res.slope * x

# Визуализация
fig, ax = plt.subplots(figsize=(6, 4))
ax.scatter(x, y, label='Данные')
ax.plot(x, y_pred, 'r--', label='Регрессия')
ax.legend()
plt.show()
```

## Реализация через sklearn

```python
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

X = x.reshape(-1, 1)   # sklearn ожидает матрицу
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

model = LinearRegression()
model.fit(X_train, y_train)

y_pred = model.predict(X_test)

print(f"Коэф. b₀ (intercept) = {model.intercept_:.4f}")
print(f"Коэф. b₁ (slope)     = {model.coef_[0]:.4f}")
print(f"MSE  = {mean_squared_error(y_test, y_pred):.6f}")
print(f"RMSE = {mean_squared_error(y_test, y_pred)**0.5:.6f}")
print(f"R²   = {r2_score(y_test, y_pred):.4f}")
```

## Градиентный спуск

Когда аналитическое решение невозможно (матрица необратима, данных слишком много), применяют **градиентный спуск**.

**Идея:** двигаться в сторону, противоположной градиенту функции потерь, с шагом `η` (learning rate):

```
xₙ = xₙ₋₁ − η · f'(xₙ₋₁)
```

**Критерии остановки:**
- Малая длина градиента: `||∇f|| ≤ ξ`
- Малое изменение шага: `|xₙ − xₙ₋₁| ≤ ξ`
- Малое изменение функции: `|f(xₙ) − f(xₙ₋₁)| ≤ ξ`

**Проблемы:**
- Большой `η` → перешагивание минимума / взрыв градиента.
- Малый `η` → медленная сходимость / преждевременная остановка.
- Локальные минимумы при невыпуклых функциях.

```python
from sklearn.linear_model import SGDRegressor

sgd = SGDRegressor(max_iter=1000, eta0=0.01, random_state=42)
sgd.fit(X_train, y_train)
print(f"R² = {r2_score(y_test, sgd.predict(X_test)):.4f}")
```

## Регуляризация

Предотвращает переобучение, штрафуя за большие веса:

```
Q_regul = Q_train(β) + λ · R(β)
```

| Тип | Название | Формула R(β) | Эффект |
|-----|---------|-------------|--------|
| **L2 (Ridge)** | Гребневая | Σβᵢ² | Уменьшает все веса |
| **L1 (Lasso)** | Лассо | Σ|βᵢ| | Обнуляет часть весов → отбор признаков |

```python
from sklearn.linear_model import Ridge, Lasso

ridge = Ridge(alpha=1.0)
ridge.fit(X_train, y_train)

lasso = Lasso(alpha=0.1)
lasso.fit(X_train, y_train)
# Нулевые коэффициенты — признак не важен
print(lasso.coef_)
```

## Переобучение и кросс-валидация

```python
from sklearn.model_selection import KFold, cross_val_score

kf = KFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(LinearRegression(), X, y, cv=kf, scoring='r2')
print(f"CV R² = {scores.mean():.3f} ± {scores.std():.3f}")
```

**Переобучение:** модель отлично работает на train, но плохо на test.
Признак: высокие веса β, большое расхождение MSE_train и MSE_test.

## Допущения и проверки

| Допущение | Тест |
|-----------|-----|
| Нормальность остатков | Шапиро-Уилк (`scipy.stats.shapiro`) |
| Отсутствие автокорреляции | Дурбин-Уотсон (`statsmodels`) |
| Гомоскедастичность | Голдфельд-Квандт |
| Нет мультиколлинеарности | Корреляционная матрица, VIF |

```python
# Шапиро-Уилк
from scipy import stats
stat, p = stats.shapiro(y - y_pred)
print("Нормальность остатков:", "да" if p > 0.05 else "нет")

# Дурбин-Уотсон
from statsmodels.stats.stattools import durbin_watson
dw = durbin_watson(y - y_pred)
print(f"DW = {dw:.3f}")   # ~2 → нет автокорреляции
```

## Краткий итог

| Концепция | Суть |
|-----------|-----|
| МНК / OLS | Минимизация суммы квадратов отклонений |
| Градиентный спуск | Итерационный поиск минимума |
| R² | Доля объяснённой дисперсии (чем ближе к 1, тем лучше) |
| Ridge / Lasso | Регуляризация против переобучения |
| Кросс-валидация | Объективная оценка обобщающей способности |

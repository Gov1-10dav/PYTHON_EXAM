**Решающее дерево (Decision Tree)** - это модель машинного обучения, которая использует древовидную структуру для принятия решений на основе признаков объектов.

**Структура решающего дерева:**
- Корневой узел - начальный узел, с которого начинается разделение данных
- Внутренние узлы - принимают решения на основе значений признаков
- Листовые узлы - содержат конечный результат (класс или значение)

**Принцип работы:** алгоритм рекурсивно разделяет данные по наиболее информативным признакам. На каждом шаге выбирается признак и пороговое значение, которые лучше всего разделяют данные.

**Критерии разделения:**
- Энтропия - мера неопределенности в данных
- Индекс Джини - мера чистоты узла
- Дисперсия (для задач регрессии)

**Преимущества:** интерпретируемость, простота визуализации, не требует масштабирования данных.
**Недостатки:** склонность к переобучению, чувствительность к шуму в данных, нестабильность (небольшие изменения в данных могут привести к совершенно другому дереву).

**Случайный лес (Random Forest)** - ансамблевый метод, который строит множество решающих деревьев и объединяет их результаты.

**Принцип построения случайного леса:**
1. Для каждого дерева из обучающей выборки формируется подвыборка (бутстрэп-выборка)
2. На каждом узле дерева рассматривается случайное подмножество признаков
3. Деревья строятся до полного разрастания (без обрезки)
4. Для классификации используется голосование, для регрессии - усреднение

**Преимущества случайного леса:**
- Высокая точность и устойчивость к переобучению
- Возможность оценки важности признаков
- Хорошо работает с большими размерностями
- Устойчив к выбросам и пропущенным значениям
- Не требует тщательной настройки гиперпараметров

**Недостатки:** меньшая интерпретируемость по сравнению с отдельным деревом, требование к памяти и времени обучения, сложность объяснения отдельных предсказаний.

**Важность признаков** в случайном лесе оценивается по тому, как часто признак используется для разделения в узлах и насколько эти разделения улучшают чистоту узлов.
